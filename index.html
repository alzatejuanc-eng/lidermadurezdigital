Conectar hoja de cálculo

Elige dónde guardar los datos de cada evaluación

Recomendado para empezar. Gratis, sin cuentas adicionales. Usa tu Google Drive existente. Los datos llegan a tu hoja en segundos.

Configuración paso a paso

  1. Crea una hoja de cálculo en Google Sheets
    Ve a sheets.google.com → Nuevo → Hoja en blanco. Nómbrala "MIEDE Evaluaciones".
  2. Abre el editor de Apps Script
    En la hoja, ve a Extensiones → Apps Script. Borra el código existente y pega el siguiente:
function doPost(e) {
  try {
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName("Evaluaciones") || ss.getActiveSheet();
    var data = JSON.parse(e.postData.contents);
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(data.headers);
      sheet.getRange(1,1,1,data.headers.length)
           .setFontWeight("bold").setBackground("#2a2a36")
           .setFontColor("#ffffff");
    }
    sheet.appendRow(data.row);
    return ContentService
      .createTextOutput(JSON.stringify({status:"ok"}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:"error",msg:err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
  1. Despliega como aplicación web
    Clic en Implementar → Nueva implementación. Tipo: Aplicación web. Ejecutar como: Yo mismo. Acceso: Cualquier persona. Clic en Implementar y Autorizar.
  2. Copia la URL de implementación
    Aparece una URL que empieza con https://script.google.com/macros/s/... — cópiala y pégala aquí:

Airtable es una base de datos con interfaz de hoja de cálculo. Tiene plan gratuito generoso. Ideal si quieres filtrar, buscar y visualizar las evaluaciones de forma avanzada.

Configuración paso a paso

  1. Crea una cuenta gratuita
    Ve a airtable.com → Sign up. El plan gratuito permite hasta 1,000 registros por base.
  2. Crea una nueva Base
    Clic en "+ Add a base" → Desde cero. Nómbrala "MIEDE Evaluaciones". Renombra la primera tabla a "Respuestas". Los campos se crean automáticamente al guardar el primer registro.
  3. Obtén tu API Key (Personal Access Token)
    Ve a airtable.com/create/tokens → Create new token. Nombre: "MIEDE". Scopes: data.records:write. Access: tu base. Copia el token generado (empieza con pat...).
  4. Obtén el ID de tu base
    Abre tu base → clic en el icono de ayuda (?) → API Documentation. Verás tu Base ID que empieza con app....

Supabase es una base de datos PostgreSQL gratuita. La opción más robusta. El plan gratuito incluye 500MB, 50,000 filas y sin límite de tiempo.

Configuración paso a paso

  1. Crea una cuenta y proyecto
    Ve a supabase.com → New project. Elige una contraseña segura y guárdala. Espera ~2 minutos a que el proyecto inicie.
  2. Crea la tabla con este SQL
    Ve a SQL Editor en el panel izquierdo y ejecuta:
CREATE TABLE evaluaciones ( id bigint generated always as identity primary key, created_at timestamptz default now(), nombre text, email text, telefono text, pais text, edad int, cargo text, genero text, ecosistema text, q1 int, q2 int, q3 int, q4 int, q5 int, q6 int, q7 int, q8 int, q9 int, q10 int, q11 int, q12 int, q13 int, q14 int, q15 int, q16 int, q17 int, q18 int, q19 int, q20 int, q21 int, q22 int, q23 int, q24 int, q25 int, q26 int, q27 int, q28 int, q29 int, q30 int, q31 int, q32 int, score_total int, porcentaje int, nivel text, titulo text, p1 int, p2 int, p3 int, p4 int, p5 int, p6 int, p7 int, p8 int, fortaleza text, brecha text );
  1. Habilita el acceso anónimo de inserción
    Ve a Authentication → Policies, selecciona la tabla evaluaciones → New Policy → Get started quickly → elige "Enable insert access for all users" → Save.
  2. Obtén tus credenciales
    Ve a Project Settings → API. Copia la Project URL y la clave anon public.

Sin configuración necesaria. Al terminar cada evaluación, podrás descargar un archivo CSV que abre perfectamente en Excel, Google Sheets o cualquier hoja de cálculo.

Esta opción siempre está disponible en la pantalla de resultados, independientemente de si tienes configurada una base de datos o no.

El archivo CSV incluye:

  • ✓ Datos del participante (nombre, email, país, etc.)
  • ✓ Las 32 respuestas de la evaluación (valores 1-5)
  • ✓ Score total, porcentaje y nivel de madurez
  • ✓ Score individual de cada uno de los 8 pilares MIEDE 2.0
  • ✓ Fortaleza principal y brecha a resolver
